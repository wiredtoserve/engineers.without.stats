The following directories are contained in this folder:
1. Source code
2. Notebooks
3. Demo

1. The source code directory contains the following files:
- common.py: a collection of common functions used throughout the project
- train_classifier.py: The script used to define and train our classifier
- Files beginning with gen_.py are scripts used to compute the various features
  of the described in the report.
- Files beginning with plot_.py are used to generate the figures used in the
  report

2. The Notebooks directory contains exploratory code used in this report

3. The demo folder contains an interactive Jupyter notebook for estimating the
MBTI score from a sample piece of text.
